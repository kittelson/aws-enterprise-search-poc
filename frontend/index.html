    • What is KODEX?

    KODEX is a generative AI-powered Traffic engineering knowledge Agent that can assist users with by answering a natural language question. In this case, we are equipping the open-source LangChain ConversationalAgent with tools that include an Anthropic Claude LLM hosted on Amazon Bedrock using documents and data stored on Amazon DynamoDB and Amazon Kendra.


    How can I use the AI?

    You begin by clicking the chat bubble at the bottom-right of the screen.

    The project is predicated on a fork from Kyle Blocksom (AWS). His blog on the project is found here
